# Awareness Chatbot

## Project Description
This project is a console-based chatbot developed using C#. 
The chatbot interacts with the user and provides awareness information through a conversation system.

The program also includes a voice greeting and ASCII art to improve the user experience.

## Features
- Chatbot conversation system
- Voice greeting using a WAV audio file
- ASCII art display
- Console-based user interface

## Technologies Used
- C#
- .NET
- NAudio Library

## Project Files
Program.cs – Main entry point of the application  
Chatbot.cs – Handles chatbot conversation logic  
UI.cs – Handles the user interface  
AudioPlayer.cs – Plays the WAV greeting sound  
POE.wav – Voice greeting file  
ascii.txt – ASCII art displayed in the program  

## How to Run the Program
1. Open the project in Visual Studio.
2. Build the solution.
3. Run the program.
4. The chatbot will greet the user with audio and ASCII art.
5. The user can interact with the chatbot through the console.
